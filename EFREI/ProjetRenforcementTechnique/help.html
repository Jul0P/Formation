Comment Jouer

Règles du Jeu

Le Multitask Game est un jeu de survie où vous devez gérer plusieurs mini-jeux simultanément. Votre objectif est de survivre le plus longtemps possible et d'obtenir le meilleur score.

Progression

  • 0-24 points : Jeu Dodge (Esquive) uniquement
  • 25-49 points : 2 jeux actifs (Dodge + Jeu 2)
  • 50-74 points : 3 jeux actifs (Dodge + Jeu 2 + Jeu 3)
  • 75+ points : 4 jeux actifs (Mode Chaos Complet)

Vies et Game Over

Vous commencez avec 3 vies. Perdre toutes vos vies entraîne un Game Over. Votre score est sauvegardé dans le classement si il figure parmi les 10 meilleurs.

Contrôles

Jeu Dodge (Esquive)

  • Flèche Gauche (←) : Déplacer vers la gauche
  • Flèche Droite (→) : Déplacer vers la droite

Contrôles Généraux

  • N'importe quelle touche : Commencer une nouvelle partie
  • Pause : Le jeu se met automatiquement en pause si vous changez d'onglet

Note : Les contrôles pour les jeux 2, 3 et 4 seront dévoilés lorsque vous les débloquerez !
