Comment Jouer

Règles du Jeu

Le Multitask Game est un jeu de survie où vous devez gérer plusieurs mini-jeux simultanément. Votre objectif est de survivre le plus longtemps possible et d'obtenir le meilleur score.

Progression

  • 0-24 points : Dodge - Flèches ← →
  • 25-49 points : Dodge + Cube Game - Z,Q,S,D
  • 50-74 points : Dodge + Cube + Math Game - Clavier numérique
  • 75+ points : Mode Chaos Complet - Barre Espace

Vies et Game Over

Vous commencez avec 3 vies. Perdre toutes vos vies entraîne un Game Over. Votre score est sauvegardé dans le classement si il figure parmi les 10 meilleurs.

Contrôles

Jeu Dodge (Esquive) - Score 0+

  • Flèche Gauche (←) : Déplacer vers la gauche
  • Flèche Droite (→) : Déplacer vers la droite
  • Objectif : Esquiver les flèches qui tombent

Cube Game (Mouvement) - Score 25+

  • Z : Déplacer vers le haut
  • Q : Déplacer vers la gauche
  • S : Déplacer vers le bas
  • D : Déplacer vers la droite
  • Objectif : Collecter les zones avant qu'elles expirent (10 secondes)

Math Game (Calcul) - Score 50+

  • Clavier numérique (0-9) : Saisir la réponse
  • Objectif : Résoudre les opérations (+, -, ×) en moins de 10 secondes
  • Note : Validation automatique à la saisie

Flappy Game (Physique) - Score 75+

  • Barre Espace : Faire voler la balle
  • Objectif : Naviguer entre les obstacles sans les toucher
  • Note : Toucher le sol ou le plafond ne fait pas perdre de vie
