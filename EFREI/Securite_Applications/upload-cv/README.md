# Upload CV

## Installation

1. Cloner le projet.
2. Vérifier que le dossier `uploads/` existe à la racine et est accessible en écriture par PHP (chmod 0755 ou 0775 selon l'hébergement).
3. Lancer un serveur local depuis le dossier `public/` :

   php -S 0.0.0.0:8000

4. Ouvrir http://localhost:8000
